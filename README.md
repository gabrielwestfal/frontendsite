 AgroWestfal 

Projeto desenvolvido para a disciplina do **1º período** do curso de **Sistemas para Internet** do Instituto Federal do Espírito Santo.

O objetivo do projeto é criar uma landing page para a empresa fictícia **AgroWestfal**, aplicando conceitos básicos de desenvolvimento web utilizando HTML e CSS.

 Tecnologias utilizadas

* HTML5
* CSS3
* Bootstrap 5

 Objetivos do projeto

* Praticar estruturação semântica em HTML
* Aplicar estilização com CSS
* Desenvolver layouts responsivos
* Trabalhar organização visual e identidade de marca
* Utilizar componentes básicos do Bootstrap

 Funcionalidades

* Seção Hero
* Sobre a empresa
* Serviços oferecidos
* Diferenciais
* Depoimentos
* Call To Action (CTA)
* Formulário de cadastro/contato

 Prévia do projeto

Landing page voltada para o setor do agronegócio, com foco em design moderno e responsivo.
